MySQL Wiki

安装

emerge --ask dev-db/mysql

启动服务

OpenRC

启动

rc-update mysql start

默认启动

rc-update add mysql default

Systemd

启动

system start mysql

默认启动

system enable mysql

第一次可能无法启动

使用emerge --config生成配置文件

emerge --config dev-db/mysql

根据要求配置即可

root用户密码在这一步设置

增加安全性(可选)

mysql_secure_installation

比如增加密码时间有效性

基础使用

# 连接数据库
mysql -u root -p

安装工具

mycli：

emerge --ask --verbose mycli

Examples:

mycli my_database
mycli -u my_user -h my_host.com my_database
mycli mysql://my_user@my_host.com:3306/my_database

连接数据库

局域网内连接报错：由于目标计算机积极拒绝，无法连接

修改/etc/mysql/my.cnf文件：

[mysqld]
bind-address = *

隐藏历史记录

将保存有mysql命令的文件链接到/dev/null，置入黑洞之中。

rm ~/.mysql_history
ln -sf /dev/null ~/.mysql_history
