升级内核

建议在make.conf中添加USE='symlink'，然后更新系统。

sudo emerge --ask --verbose --update --deep --newuse @world

Configuration

sudo make menuconfig

Build

sudo make -j4

Install

sudo make modules_install
sudo make install

BootLoader

sudo grub-mkconfig -o /boot/grub/grub.cfg

重启以应用新的内核。

Clean

sudo eclean-kernel -n 1

仅保留当前内核。
