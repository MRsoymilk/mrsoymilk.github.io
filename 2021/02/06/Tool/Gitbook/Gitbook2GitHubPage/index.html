  • gitbook

使用方法

创建两个分支，一个用来开发，一个用来网页展示

开发分支为main
网页展示为book

操作

  1. 在main分支上生成静态页面
gitbook build

默认在_book目录中生成。

  1. 创建分支book：
git branch book
  1. 切换分支book：
git checkout book

删除所有文件
将_book中文件转移至根目录

  1. 在不同分支下git push

可能需要执行--set-upstream指令

git push --set-upstream origin book

登录Github相应项目 -> Settings -> GitHub Page -> 选择相应分支
