写在前面

Windows 10 系统电脑桌面图标全部消失，右击只有显示桌面图标，然而并没有什么用。桌面不能点点点可怎么活，还好重启Windows资源管理器能解决。

解决方法

*启动任务管理器: *
Ctrl + Alt + Delete 勾选任务管理器
任务栏右击选择任务管理器
