写在前面

操作系统：Windows10 64 位

IDE：Intellij IDEA Ultimate Version 2019.1.3

其他：Tomcat，安装 JDK(如果没有安装，IDEA 会在创建项目的时候提醒你去安装)

开始

  1. 新建一个项目

    创建 Web Application

  2. 命名你的项目

  3. 添加 Tomcat 配置

  4. 一些配置

    • 为你的配置取名
    • 选择本地安装的 Tomcat 服务器
    • 其他保持默认即可，如有需要自行配置
  • 选择部署位置
  1. 添加测试文件index.jsp

  • 解决Cannot resolve method 'println(java.lang.String)'
    • File -> Project Structure -> Libraries 添加 lib 文件
    • 需要添加的 lib 文件: tomcat 安装位置/lib
    • File -> Project Structure -> Modules 添加 lib 包
    • ✔ 你的添加
    • 稍等一会儿即可解决
    • 不添加也能运行，但是添加后有代码补全和提示！👍
  1. 启动你的服务

  1. 启动成功，自动跳转到网页。
