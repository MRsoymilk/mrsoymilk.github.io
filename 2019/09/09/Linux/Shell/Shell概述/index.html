#!是什么？

Shebang（也称为 Hashbang ）是一个由井号和叹号构成的字符序列 #!，其出现在文本文件的第一行的前两个字符。 在文件中存在 Shebang 的情况下，类 Unix 操作系统的程序载入器会分析 Shebang 后的内容，将这些内容作为解释器指令，并调用该指令，并将载有 Shebang 的文件路径作为该解释器的参数

/bin/bash
使用的 shell 解释器，绝对路径
种类繁多，常用的为 bash
